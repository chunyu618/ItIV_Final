# ItIV_Final
Final Project of Introduction to Intelligent Vehicle, NTU CSIE

- `bus.py`: parse data
- `MDP.py`: Use `pulp` to find optimal solution. The result is used in `simulation.py`
- `simulation.py`: simulation code.
